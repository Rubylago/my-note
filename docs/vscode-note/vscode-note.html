設定預設終端機​

使用快速鍵 Ctrl + Shift + P 叫出選單
輸入 Terminal: Select Default 應該會跑出 終端機:選取預設設定檔

點下去!
會跳出你目前有的shell選項，選想要的預設終端機就完成啦~

現在可以用快速鍵 Ctrl + Shift + 反引號 把終端機叫出來，應該會出現剛剛選的預設選項了!
